This test is used to showcase the EFDS using 2000 samples. The 2000 samples are split evenly between positive and negative samples. The configuration from the classifier are contained and derived from the SC_stages_3_boostRnds_1_featureIncr_50.bin file. The output is contained in the out.txt file and dispalys the number of true and false positives. The diagnostics.txt file shows timing information as well as displays the datacycle of every sample.
